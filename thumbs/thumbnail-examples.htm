Thumbnail examples

Resoluations and aspectratio

This page demonstrates several ways to retrieve thumbsnails for an indexed webcast. Thumbnails are small images captured from the video stream at certain points. These thumbnails are available at the following sizes:

  • Original size
  • 320x240
  • 160x120

The 320x240 and 160x120 resolution thumbnails are in 4:3 aspect ratio and if the source is in another aspectratio like for example 16:9 then these frames will be cropped to 4:3 (pan & scan method). If the original aspect ratio is required then you use must the original format but keep in mind that the resolution of this thumb is the same as the source.

Retrieving a thumb at a specific location

There are several ways to retrieve a thumbnail:

  • Default image
  • By time offset
  • By actual time
  • By percentage

The default image is always available and the other types are only available for registered users. This is only of interest when you have webcasts which require user input like for example a password or e-mail address. This is to protect the content to be viewed for users that are not authorized to view such content.
